R1.02 - HTML / CSS - TP 2

Dans ce TP vous allez créer votre première feuille de style CSS. Elle ne s'occupe pour l'instant que des aspects basiques de la présentation : les fonts, les tailles, les couleurs, et quelques positionnements.

  1. Mise en place
    1. Historisation

      1. Dans l'explorateur de fichiers de windows, recopiez le répertoire créé lors du tp précédent, puis renommer le tp2.

      2. Vérifiez bien que ce nouveau répertoire contient une copie de votre fichier index.html, ainsi que de l'image que vous avez utilisée au tp1.

    2. Création du sous-répertoire et du fichier CSS

      1. Dans le répertoire tp2, créez un répertoire styles. Dans celui-ci, ajoutez un fichier index.css, avec la même méthode de création et de renommage que pour index.html.

      2. Si ce n'est pas déjà fait, associez les fichiers .css avec Notepad++ (via le menu contextuel et Propriétés, comme vu précédemment)

    3. Intégration du css

      1. Ouvrez le fichier css et placez en première ligne, la déclaration de l'encodage : @charset "utf-8";

      2. Ouvrez le fichier html et ajoutez aux fils de l'élément head, le lien vers le fichier css :
        <link rel="stylesheet" href="./styles/index.css" />

    4. Test et Validation

      1. Pour tester la mise en place, ajouter une règle très visible dans le fichier css, comme par exemple : *{color:red;}

      2. Vérifiez l'indentation du fichier html

      3. Validez le document html

      4. Validez la feuille css

      5. Visualisez

  2. Formatage basique

    (avant toute chose, enlevez ce vilain rouge)

    1. Place Nette

      1. En toute première règle, utilisez la règle générique pour annuler les choix d'espacements par défaut du navigateur.

    2. Entête

      1. Détachez le bloc header en lui donnant une marge (dans les quatre directions) [1 seule règle]

        Donnez une couleur de fond à ce bloc [1 seule règle]

      2. Centrez les écritures des fils du bloc header [1 seule règle]

      3. Agrandissez nettement la première ligne, et elle seulement [1 règle avec sélecteur composé]

        Agrandissez un peu moins la seconde ligne, et elle seulement [1 règle avec sélecteur composé]

    3. Encadré

      1. Encadrez le bloc aside avec une bordure (noire) ; séparez un peu son contenu du bord ; séparez-le aussi du reste [1 règle ; 2 propriétés]

      2. Suprimez l'élément img dans le fichier html. A la place, dans le fichier css, utilisez l'image comme fond de l'élément aside.

        Faites en sorte que l'image n'apparaisse qu'en un seul exemplaire, qui couvre tout le fond, éventuellement sans que toute l'image ne soit visible. [1 règle (existante) ; 3 propriétés]

      3. Le texte n'est plus lisible. Changez sa couleur, et éventuellement sa taille. Remarquez qu'en ajoutant un ombrage(shadow) contrastant et légèrement flouté, on améliore encore la lisibilité. [1 sélecteur composé ; 3 propriétés]

    4. Contenu Principal

      1. Agrandissez les titres de premier niveau, et soulignez-les. Détachez les un peu du bord gauche [1 règle (2 possibilités de spécificité) ; 3 propriétés]

      2. Détachez un peu les articles les uns des autres [1 règle (plusieurs spécificités possibles) ; 1 propriété "racourci"]

      3. Agrandissez les titres de second niveau, mais moins que ce de premier niveau, et soulignez-les. [1 règle (plusieurs spécificités possibles) ; 2 propriétés]

      4. Détachez un peu les paragraphes les uns des autres, et indentez la première ligne de chacun. [1 règle (plusieurs spécificités possibles) ; 2 propriétés]

    5. Pied de page

      1. Donnez une couleur de fond au pied de page

      2. Alignez le texte à droite, et centrez l'adresse

      3. Ecartez les contenus du pied de page du bord, tout en maintenant l'effet "bandeau" (i.e. la couleur de fond du pied de page couvre toute la largeur de la page)

  3. Validez votre fichier css (voir lien ci-dessus)
